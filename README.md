# My JSFiddles
These are random pages, toys, tools... I published from 2023 onwards [on jsfiddle.net](https://jsfiddle.net/u/arian_/fiddles/).

I realized a while ago that I should really move them off the site, so here they are.

### Archival
In `_archiving`, you will find single-use scripts that I "made" to collect all versions into a repo, and generate the index.html and its thumbnails.

I guess it could be useful to anybody who also had a cacophony of tools they had stuck on JSFiddle, as long as scripts specific to me (Safari History.db extraction and mentions of "arian_") get modified. It may be good to make this usable as a general-purpose "JSFiddle to Git with history" utility.
OR, JSFiddle could just provide this feature themselves instead of being jerks and [not even responding to help](https://github.com/jsfiddle/jsfiddle-users/issues/1782#issuecomment-3708484101) with it.

### Future
For now, this repo is just meant as an archive. Later I should find a better replacement for JSFiddle, since it's currently the least worst solution if I want to just vibe code a certain tool and have it "just work".

Maybe it'll still be a development tool, then the finished products go onto my site or as features added to a bigger project. I don't know yet.

The scripts I've made for this are capable of auto-updating, but...
